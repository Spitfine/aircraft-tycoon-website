# Aircraft Tycoon Website

Official website repository for **Aircraft Tycoon**, a historical industrial, technological, and economic management game developed by **Spitfine**.

The game is currently in **Early Alpha**. The first complete gameplay cycle is functional, from founding the company and designing a pioneering aircraft through testing, market demonstration, first order, production, delivery, payment, and reputation impact.

## Deployment

This static website is intended to be deployed through Azure Static Web Apps from the `main` branch.

Public contact: `contact@aircraft-tycoon.com`
